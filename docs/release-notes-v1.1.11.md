# Firestone2Green v1.1.11 更新日志

## 修复

- 修复部分用户安装“持续修复”后，每次开机都会弹出一个长期停留的管理员 PowerShell / Windows Terminal 窗口的问题。
- 持续监听任务现在通过隐藏 VBS 启动器拉起，不再显示控制台窗口。
- 安装新版持续修复前，会自动清理旧版可见监听进程，避免旧任务残留。
- “移除持续修复”现在会同时结束旧版 WatchFirestone 监听进程，并删除隐藏监听启动器文件。

## 给已安装旧版本持续修复的用户

升级到本版本后请执行一次：

1. 打开 Firestone2Green。
2. 点击“移除持续修复”。
3. 再点击“安装持续修复”。
4. 重启电脑验证：不应再弹出 PowerShell 窗口。

如果窗口已经弹出，可以先直接关闭该窗口；新版安装/移除流程会清理旧监听。
