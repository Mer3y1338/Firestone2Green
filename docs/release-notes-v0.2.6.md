# Firestone2Green v0.2.6 更新日志

发布日期：2026-07-19

> 本版本保持现有主界面和功能布局不变，新增首次启动免费软件确认，进一步减少被倒卖和误付费的情况。

## 首次启动确认

- 首次启动时显示免费开源声明。
- 必须输入完整文字 `免费软件 拒绝付费` 后，**启动程序**按钮才会启用。
- 在确认输入框右侧新增**复制**按钮，可一键复制需要输入的确认文字。
- 点击关闭窗口或**退出程序**会直接退出，不会进入主界面。
- 确认记录按用户保存在 `%LOCALAPPDATA%\Firestone2Green\disclaimer.v0.2.6.ok`，同一台电脑确认一次即可。
- 0.2.5 及更早版本的旧提示记录不会跳过本版本的新确认。

## 启动与版本

- 版本更新至 `0.2.6`。
- 保持 WPF + .NET 10、单 EXE、现有视觉风格和响应式布局。
- 保留上一版本已经验证的标准 automation 启动方式：优先 `OverwolfLauncher.exe`，必要时使用 `Overwolf.exe` 作为同参数备用。
- 保留 0.2.5 的环境检测、管理员权限提示和运行日志修复。

## 发布文件

- `Firestone2Green_v0.2.6.exe`：单 EXE 主程序。
- `Firestone2Green_v0.2.6_dist.zip`：单 EXE、README、使用教程、更新日志和许可证。
- GitHub Release 页面会同时提供源码归档。
